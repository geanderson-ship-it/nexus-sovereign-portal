# **App Name**: SkillRise Academy

## Core Features:

- Listagem de Cursos: Exibir os cursos de treinamento disponíveis com detalhes como preço, duração e programa.
- Autenticação de Usuário: Contas de usuário seguras com funcionalidade de login/registro.
- Pesquisa e Filtragem: Permitir que os usuários encontrem facilmente cursos com base em palavras-chave, categoria e nível de habilidade.
- Recomendações Personalizadas: Recomendações de cursos orientadas por IA com base nos perfis de usuário e histórico de aprendizado, usando uma ferramenta LLM para classificar os cursos relevantes.
- Carrinho de Compras e Checkout: Permitir que os usuários adicionem cursos a um carrinho e concluam transações de compra seguras.
- Integração de Gateway de Pagamento: Integre-se com provedores de pagamento (por exemplo, Stripe, PayPal) para processar pagamentos de cursos.
- Avaliações e Ratings de Clientes: Permitir que usuários registrados avaliem e comentem sobre os cursos de treinamento.

## Style Guidelines:

- Cor primária: Azul profundo (#3F51B5), evocando confiança, autoridade e experiência em desenvolvimento profissional.
- Cor de fundo: Azul muito claro (#E8EAF6), mantém uma aparência profissional, permite que o conteúdo se destaque.
- Cor de destaque: Violeta (#7E57C2) para destacar as principais interações e criar uma sensação de sofisticação.
- Fonte do título: 'Poppins', sem serifa, oferece um estilo geométrico e moderno.
- Fonte do corpo: 'PT Sans', sem serifa, para proporcionar legibilidade e calor.
- Ícones profissionais e claros que representam diferentes categorias e recursos de treinamento.
- Transições e animações sutis para uma experiência de usuário suave ao navegar pelos cursos e páginas.
